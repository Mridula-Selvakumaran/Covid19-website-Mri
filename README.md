# COVID-19 Data Exploration Dashboard

An interactive **Streamlit** web application for exploring global COVID-19 trends. The dashboard visualizes daily cases, weekly vaccinations, and monthly cases by continent, along with a snapshot of the top 10 countries based on total reported cases.  

---

## Features

- **Daily Cases View:**  
  Compare daily new cases across selected countries.

- **Weekly Vaccinations View:**  
  Visualize vaccination progress week by week.

- **Monthly Cases by Continent:**  
  Track aggregated monthly case trends across continents.

- **Top 10 Countries Snapshot:**  
  View a comparative summary of the most affected countries using the latest available data.

---

## Data Source

All data is sourced directly from [Our World in Data - COVID-19 Dataset](https://covid.ourworldindata.org/data/owid-covid-data.csv), which is updated frequently and publicly available.

---

## Tech Stack

- **Frontend / Framework:** Streamlit  
- **Data Handling:** Pandas  
- **Visualization:** Matplotlib
